Vika 1-2

The subject of these first two weeks was simply the creation of this website. This page will cover most of the tasks that where done.

Part 1 - The creation of the website

The first thing I did knowing I needed this website, was finding a nice and free template to ease my work. The template was found on the website Colorlib.com and it was completely Free The second thing I did was to download it and open it with my website environment, Visual Studio Code. I had already used it in another website course in the University of Iceland. So I had some experience. Then I did some rememorization about web design to get an idea about how I should make my website. I scanned through all the websites the template had to offer. Then I desided what sites of the template I could use and for what. After doing that I made a sketch of how my website should be designed. I desided to have 5 main sites. The Introduction about me, my resume, my University courses, how to contact me. And finally a special site for the course manufacturing. And on this site you could go into the diary and more which still hasn't been desided. The template was good and easy to understand for someone who already had learned the basics of html, css and javascript. However; it needed some features and some had to be removed. E.g. the resume had some extra features which I didn't like. Also there was no button on the Manufacturing site, so there was no proper conection to that site and the blog, i.e. the diary and the home assignments. I did some css for that, I also tried to make so javascript for some graphic and also to publish multiple different blogs under the same html file. This could be done by creating an empty html file for the diary and make a js file which would read a JSON string once it would be opened. The JSON string depended on which diary was clicked. In the end I decided rather simply to make many html files. One for each diary.

Part 3 - Upploading the website

When creating the website I used git bash a lot. I used a few commands, git add file.html, git commit -m "message", and git push. Github also hosted the website for me. That was incredibly easy. Simply three clicks after adding the files to my repository.

Part 5 - The Final Project

The final project will be a group project. The group was supposed to be comprised of three people. Which was very convenient for us. The group members are me (Viktor Helgi Gizurarson), Lovísa Þórunn Harðardóttir, and Kristján Gíslason. I have proposed to my group that we should find an idea for our final project as soon as possible. Because if we decide now what we should do then we can plan all of the other home assignments we will do around the final project. We have got some ideas. However, we have not decided which to carry out. Also the group will need to do another website, and I have proposed to do it because I believe we should realise who as got which skils and divide the work accordingly.
