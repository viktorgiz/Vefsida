Hi There

My name is Viktor Helgi Gizurarson and I created this website in a course called Manufacturing Processes, or in Icelandic Framleiðsluferli. This website includes many things, for example the course of my education and those jobs that I have had. However, as for now, the main subject this website will address is the course Manufacturing Processes. This will include all the assignments made through the course and a diary where most of the activities associated with the course will be logged.

This website is split up into five pages. The first one, University of Iceland, where you can see all the classes I have undertaken. Second, Resume, which includes all the course of my employment. Third, which is this page and which is simply an introduction to this website. The fourth page is called Manufacturing and this page is the main page for the course. It consists of three sections, information about the group project which soon will begin and a preface of the diary and the assignments which, as of january 2020, have not been finished. Finally, the last page includes information on how you can contact me. The development of this website is still in progress and therefore the last pages does not yet properly function. Anyhow you can contact me through mail, vhg6@hi.is.
